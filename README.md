# 🌾 Marketplace de Productos Agrícolas API

API RESTful desarrollada con FastAPI para E-Commerce y Retail agrícola.

## Funcionalidades

- CRUD de productos
- Filtros avanzados
- Paginación
- Ordenamiento
- Búsqueda
- Upload de imágenes
- Documentación Swagger automática

## Ejecutar

uvicorn main:app --reload
